   1. What is Semantic HTML?
Semantic HTML is used to reinforce the meaning of the information in webpages, making it more interesting that simple HTML.

    2. What is HTML used for?
HTML is used to create web pages, and is used as the framework of a webpage allowing you to put information in an organized way. 

    3. What is an attribute and where do we put it?
 And attribute is something we assign to an element to make it do something specific, such as “href” and “src”.

    4. What is the h1 tag used for? How many times should I use it on a page?
The h1 tag is used to creat a large headline style text, using it on text conveys that text’s importance. The h1 tag should only be used once, and used on the main subject of the page. 

    5. Name two tags that have required attributes
The <a> tag, and the <img> tag. 

    6. What do we put in the head of our HTML document?
The head element is where we would put the title element, or the style element.

    7. What is an id?
The id is used in html to identify a unique element that can then be pointed to in css and javascript.
 
    8. What elements can I add an id to?
The id can be used on any html element.
 
    9. How many times can I use the same id on a page? 
Only once.

    10. What is a class? 
The class attribute is similar to an id except a class can be used by multiple elements, whereas an id can only belong to one element.

    11. What elements can I add a class to? 
The class can be used on any html element as well.

    12. How many times can I use the same class on a page? 
One class can be used multiple elements.

    13. How do I get my link to open in a new tab?
When creating your anchor element, add:
`target=”_blank”`
And the link will open in a new tab.

    14. What is the alt attribute used for? 
The alt attribute is used for an img element as an alternative in case the image cannot be displayed.

    15. How do I reference an id?
To reference an id, you use  #idName

    16. What is the difference between a section and a div
The section defines sections like headers and footers and shows that content within it relates to one thing, while the div is a generic element used to group block-elements together to format them with css.

    17. What is CSS used for? 
CSS is used to style a web page, and give it a more appealing look with colors, fonts, animations, and different layouts.
